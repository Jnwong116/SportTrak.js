Getting Started

  • Include the sportTrak_styles.css file in your html page
  • Include the sportTrak.js file in your html page
  • Get the data for the sports game.
  • Create a new sportTraker and pass the data into it.
    •                         
                            const sportTracker = new sportTraker(soccerData);
  • You are all set up! Now you can call these functions on your new sportTraker object!

User Methods

SportTracker.makeCourt(scale)
  • Creates a court/field of the respective sport.
  • scale: is the scale you want the court/field to be scaled to.
  • Return: returns a div which contains the whole court.
  • Use return value to add the court to your page and position/stlye the court as you like.
SportTracker.displayPlayers()
  • Displays the lineups for the game, with each player and their jersey number
  • Also allows you to click on each player to view their stats
  • Return: returns two divs which contain the players on the bench.
  • Use the return value to decide if you want to display the benches of each team.
  • Position the benches as you wish
SportTracker.showStats()
  • Creates a table with the game stats from the game.
  • Return: returns a div which contains the whole table.
  • Use return value to add the stats table to your page and position the table as you wish.
SportTracker.playAnimation(num)
  • Plays the animation from the list of highlights.
  • num: The animation number you wish to play from the list of highlights
  • Return: NA
